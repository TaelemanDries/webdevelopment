Waarom staat de tekst in de blokken horizontaal gecentreerd?

Het wordt horizontaal gecentreerd door de text-align:center.

Ga na dat een property vertical-align:center geen effect heeft. Waarom eigenlijk niet?

Dit heeft alleen effect op inline of table-cell elementen, niet block.

Wat is de ‘simpele’ manier om een één enkele regel tekst verticaal te centreren?

De line-height even groot maken als de height. (allebei 100px bijvoorbeeld)

Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren?

Display: table-cell gebruiken samen met vertical-align: middle.

Waarom krijgt de regel voor class 'tall' voorrang?

De specifieke class selector .tall krijgt voorang op de algemene class selector .area.

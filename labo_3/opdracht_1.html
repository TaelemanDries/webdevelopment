De dom tree komt overeen met de html code van deze pagina:

HTML komt overeen met <html> van de code, HEAD, META en TITLE komen overeen met <head> van de code waar <meta charset="utf-8">en title (in dit geval: Browser default heading styles) in vermeld zijn. Dan is er nog Body met h1-h6 in en een paragraaf (p) met een referentie (a) naar de afbeelding op de pagina (html dog). De lijntjes geven aan welk element genest is in de HTML-structuur binnen een andere element.
